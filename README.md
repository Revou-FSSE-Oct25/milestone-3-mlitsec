# RevoShop - Milestone 3 (RevoU)

## Overview
RevoShop adalah aplikasi e-commerce demo menggunakan Next.js App Router untuk memenuhi tugas Milestone 3 (Module 4 dan Module 5).

## Features
- Product listing dari FakeStoreAPI
- Product detail dengan dynamic route (`/products/[id]`)
- Halaman statis (SSG): Promo dan FAQ
- Keranjang belanja dengan Context API + localStorage
- Login + middleware untuk protected route
- Checkout page (protected)
- Admin CRUD produk dengan API Routes
- ISR pada halaman admin (`revalidate = 60`)
- Unit testing (Jest + React Testing Library)

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Context API
- Jest + React Testing Library

## Run Locally
```bash
npm install
npm run dev
```

## Testing
```bash
npm test
npm run test:coverage
```
